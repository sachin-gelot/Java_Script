INTRODUCTION OF JAVASCRIPT

  1. JavaScript was invented by Brendan Eich in 1995. It was developed for NETSCAPE NAVIGATOR.
  2. JavaScript is the very important programming language. We use Java Script to create websites. No website can be created in this world without Java Script.
  3. JavaScript can dynamically modified our HTML decument.
  4. JavaScript is the case sensitive language.
  5. JavaScript is the client-side, server-side, event-based, object-oriented, interpreted programming language.
  6. JavaScript is supported by every browser infact every browser is created with the help of JavaScript. JavaScript runs inside the every browser.

we have a two ways to apply JavaScript in html

  1. External JavaScript - We create a separate file for the external JavaScript and add it to the SRC attribute inside the script tag above the body section. As soon as we define the path of that file inside.
  2. Internal JavaScript - We write internal JavaScript in the script tag above the body section.
